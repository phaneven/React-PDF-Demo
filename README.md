### React-PDF-DEMO
---

##### Major Dependencies

- React 16
- Webpack 4
- Babel 7
- Eslint
- PDF.js

##### Features

- preview PDF on mobile and pc

![screenshot](https://github.com/phaneven/React-PDF-Demo/tree/master/dist/public/pic1.jpeg)

##### Commands
- install

```
npm i
```
- development mode

```
npm run dev:client
```

- production mode (create bundles)

```
npm run build:client
```
